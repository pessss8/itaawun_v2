🗄 What is Supabase? (Read this once)

Supabase is a free online service that gives your website 3 things it currently does not have:

  • A database — stores all users, services, bookings, ratings. Think of it as an online Excel spreadsheet your website can read and write to.
  • Authentication — handles login, signup, passwords, and sessions. You will never write password security code yourself.
  • An API — your HTML pages talk to the database using a few lines of JavaScript. No PHP, no Node.js, no server needed.

Your site stays on GitHub Pages (free). Supabase sits behind it handling all the data. They talk to each other via JavaScript in your HTML files.
